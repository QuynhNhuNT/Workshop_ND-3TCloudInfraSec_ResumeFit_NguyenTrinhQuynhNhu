### Objectives of Week 3:
* Master object storage using Amazon S3 and secure data using Bucket Policies.
* Deploy relational database RDS MySQL and NoSQL database DynamoDB.
* Set up AWS Cloud9 development environment and utilize AWS CLI to manage resources.

### Tasks to implement this week:
| Day | Task | Start Date | End Date | References |

### Achievements in Week 3:
* Successfully deployed a static website on S3 and configured secure Bucket Policies.
* Deployed an RDS MySQL database in Private Subnet with secure access allowed from EC2.
* Created a DynamoDB table and performed data queries via CLI.
* Mastered basic AWS CLI command automation and cloud-based development using AWS Cloud9.
